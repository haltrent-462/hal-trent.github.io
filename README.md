## Technical Notes

## Deployment

This repo is configured for **GitHub Pages**. The site auto-publishes from the `main` branch root on every push.

To deploy elsewhere, drop `index.html` onto any static host (Netlify, Vercel, Cloudflare Pages, S3, etc.). No configuration required.
